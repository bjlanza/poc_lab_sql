# SQL Lab - Iniciar

## Un Comando

```bash
bash init
```

Instala:
- Node.js
- SQLite
- npm dependencies
- Inicializa BDs
- Crea progress file

## Arrancar

```bash
cd web
npm start
```

Abre: **http://localhost:3000**

---

## Si Algo Falla

```bash
bash check
```

Te dice qué falta.

Solución:
```bash
bash init
cd web
npm start
```
